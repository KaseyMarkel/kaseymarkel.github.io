‚ö° 5-Minute Deployment to GitHub Pages

1 Get your BMS API key

Login to BMS Pro ‚Üí Settings ‚Üí API Access ‚Üí Generate new key

2 Update index.html with your credentials

const BMS_API_CONFIG = {
  baseUrl: 'https://bmspro.io/api',
  apiKey: 'YOUR_API_KEY_HERE',
  programId: 'YOUR_PROGRAM_ID_HERE'
};

3 Create GitHub repository

Go to github.com/new and create "breeding-dashboard"

4 Push code to GitHub

git init
git add .
git commit -m "Initial dashboard"
git remote add origin https://github.com/YOUR_USERNAME/breeding-dashboard.git
git push -u origin main

5 Enable GitHub Pages

Settings ‚Üí Pages ‚Üí Source: main branch ‚Üí Save

6 Share with team!

Your dashboard is live at: https://YOUR_USERNAME.github.io/breeding-dashboard/
